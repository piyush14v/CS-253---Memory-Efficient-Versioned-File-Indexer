# Description

This program creates a word-frequency index for large text files. It does this by using incremental processing with a fixed-size buffer. The program lets you search for word frequencies, show the top-K frequent words, and compare word counts between two indexed versions.

# Compilation

Compile the program using:

g++ -O2 240748_Piyush.cpp -o analyzer

## Usage

### Word Frequency Query

./analyzer --file test_logs.txt --version v1 --buffer 256 --query word --word error

### Top-K Query

./analyzer --file test_logs.txt --version v1 --buffer 256 --query top --top 10

### Difference Query

./analyzer --file1 test_logs.txt --version1 v1 --file2 verbose_logs.txt --version2 v2 --buffer 256 --query diff --word request

## Input Arguments

--file : Path to the input file used for single-version queries.

--file1 : Path to the first input file used in the difference query.

--file2 : Path to the second input file used in the difference query.

--version : Version name associated with the file when using single-version queries.

--version1 : Version name for the first file in the difference query.

--version2 : Version name for the second file in the difference query.

--buffer : Buffer size in KB used for incremental file reading (must be between 256 and 1024).

--query : Type of query to execute. Supported values are:
          word  → returns the frequency of a word
          top   → returns the top-K most frequent words
          diff  → returns the difference in word frequency between two versions

--word : Word to search for in word or diff queries.

--top : Number of most frequent words to display in the top query.


## Implementation Notes

* File is processed incrementally using a fixed-size buffer.
* Word matching is case-insensitive.
* Object-oriented design with separate classes for file reading, tokenization, indexing, and query processing.
* Runtime polymorphism is used for different query types.
* Exception handling is used for invalid inputs and file errors.
